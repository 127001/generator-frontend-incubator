Frontend Incubator

Generate a project based on the Incentro Frontend Incubator

View on GitHub View on NPM

The Frontend Incubator is a starter kit for building scalable websites and webapplications. The developers of Incentro created this collection of best practices after trying many project structures and build processes.

Features

  • Build process with usefull Gulp tasks
  • Clear folder structure for project files
  • Generate pages with the Swig template engine
  • Build a pattern library with KSS
  • Verify javascript with jshint
  • Verify SASS with SASS lint (optional)
  • Babel for ES2015 support ('loose' mode is optional)
  • ITCSS based folder structure for SASS (optional)
  • Profit

Installation

First, install Yeoman and generator-frontend-incubator using npm (we assume you have pre-installed node.js). 

npm install -g yo
npm install -g generator-frontend-incubator

Create a project folder and run the following command from that folder to generate a new project:

yo frontend-incubator
